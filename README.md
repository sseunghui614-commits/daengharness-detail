## 7. 트러블 슈팅
문제 1. 상세페이지 이미지 경로 분기 처리 문제
*상황:
    상품 타입(C-harness / 일반 하네스)에 따라 이미지 폴더 경로가 달라
    동일한 방식으로 이미지를 불러오면 일부 상품 이미지가 출력되지 않는 문제가 발생함.
*해결:
    상품 타입을 기준으로 이미지 경로를 분기 처리하는 함수를 구현해
    상황에 맞는 이미지 경로를 동적으로 반환하도록 수정함.
*과정:
    상품 데이터 구조를 확인한 뒤,
    type 값에 따라 require 경로를 분기하는 방식으로 로직을 재정리하여
    이미지 관리 구조를 단순화함.

문제 2. 상품·리뷰 데이터 구조 불일치
*상황:
    상품 데이터와 리뷰 데이터의 필드명이 통일되지 않아 조건 분기 로직이 복잡해짐
*해결:
    리뷰 필터링 시 여러 필드명을 고려한 조건 처리 로직을 추가하여 안정적으로 대응
*과정:
    데이터 구조를 분석하고 공통 기준을 정리해 설명 가능한 코드 형태로 개선
